# Visions by Shefali — Website

Next.js site for Visions by Shefali (services, contact, testimonials, FAQ).

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy

Deploy on [Vercel](https://vercel.com/new) or any host that supports Next.js by connecting this repository.
